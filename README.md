# Hair@Home

Mobile hair stylist website for Winnipeg.

## Development

```bash
npm run serve    # Start dev server at http://localhost:1313
npm run build    # Build to docs/ folder
```

## Deployment

Push to `main` to deploy to GitHub Pages at https://reverb256.github.io/hairathome/

## Structure

```
hairathome/
├── content/           # Page content (Markdown)
├── themes/hairathome/ # Custom Hugo theme
├── static/            # Static assets
├── docs/              # Built site (deployed)
└── hugo.toml          # Hugo config
```
